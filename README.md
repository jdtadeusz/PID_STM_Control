# PID_STM_Control

**UWAGA: Projekt jest w trakcie tworzenia i nie jest gotowy do użytku!** 

## Opis

PID_STM_Control to projekt implementujący regulację PID na mikrokontrolerze STM do sterowania lewitacji piłki pinkpongowej w rurze, za pomocą wentylatora przemysłowego. 

## Status Projektu

**Projekt w przygotowaniu** 

Ten projekt jest aktualnie w fazie rozwojowej. Funkcjonalność nie jest w pełni zaimplementowana i testowana. Kod może się znacznie zmienić w przyszłości.

## Technologia

- **Język**: C 
- **Platforma**: STM Microcontroller
